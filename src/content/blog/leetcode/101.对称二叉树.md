## 二、题解

### 2.1 递归

**思路：** 一棵树轴对称，等价于它的左右子树互为镜像。定义 `check(p, q)` 判断两棵子树是否镜像：若两者都为空则对称；若只有一个为空则不对称；否则需同时满足三点——根值相等、`p` 的左子树与 `q` 的右子树镜像（外侧）、`p` 的右子树与 `q` 的左子树镜像（内侧）。从 `check(root.left, root.right)` 出发递归即可。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    /**
     * 判断一棵二叉树是否轴对称（镜像对称）
     * 思路：如果一棵树对称，那么它的左子树和右子树应该是镜像关系
     */
    public boolean isSymmetric(TreeNode root) {
        // 从根节点开始，比较它的左子树和右子树是否互为镜像
        return check(root.left, root.right);
    }

    /**
     * 递归检查两棵子树 p 和 q 是否互为镜像
     * 镜像的定义：p 的左子树要与 q 的右子树对称，且 p 的右子树要与 q 的左子树对称
     */
    public boolean check(TreeNode p, TreeNode q) {
        // 情况1：两个节点都为空，说明这一支已经对称到底了，返回 true
        if (p == null && q == null) {
            return true;
        }

        // 情况2：只有一个节点为空，另一个不为空，结构不对称，返回 false
        // （注意：这里已经排除了两者都为空的情况，所以用 || 即可）
        if (p == null || q == null) {
            return false;
        }

        // 情况3：两个节点都不为空
        // 必须同时满足三个条件才对称：
        // 1. 当前两个节点的值相等
        // 2. p 的左子树 和 q 的右子树 互为镜像（外侧对称）
        // 3. p 的右子树 和 q 的左子树 互为镜像（内侧对称）
        return p.val == q.val
            && check(p.left, q.right)   // 左对右：比较外侧
            && check(p.right, q.left);  // 右对左：比较内侧
    }
}
```

**时间复杂度**：$O(n)$

**空间复杂度**：$O(h)$
